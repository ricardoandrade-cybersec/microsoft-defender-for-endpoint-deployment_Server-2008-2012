
# Microsoft Defender for Endpoint Deployment

PowerShell scripts and deployment guidance for Microsoft Defender for Endpoint (MDE) on legacy Windows Server environments.

## Supported Platforms

- Windows Server 2012 R2
- Windows Server 2008 R2 Enterprise (Legacy)

## Features

- Administrator privilege validation
- Installation package validation
- Automated onboarding process
- Service verification
- Error handling

## Security Notice

Sensitive information has been removed from this repository.

Replace the placeholders below with values from your own environment:

```powershell
$WorkspaceID = "<WORKSPACE_ID>"
$WorkspaceKey = "<WORKSPACE_KEY>"
```

Never commit:

- Workspace Keys
- Tenant IDs
- API Keys
- Authentication Tokens

## Disclaimer

This repository is intended for educational and deployment automation purposes.
